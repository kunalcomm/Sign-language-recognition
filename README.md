##  Overview
This project is a real-time hand gesture recognition system built using Python, OpenCV, and MediaPipe.  
It detects hand landmarks through a webcam and recognizes different hand gestures based on finger positions and thumb orientation. The recognized gestures are displayed as text on the screen.

This project is created for learning and experimentation with computer vision and gesture-based interaction.

---

##  Features
- Real-time hand detection using webcam
- Accurate hand landmark detection using MediaPipe
- Gesture recognition based on finger states
- Displays detected gesture with bounding box
- Simple and beginner-friendly implementation

---

##  Supported Gestures
- Hi (Open Hand)
- Thumbs Up 
- Thumbs Down 
- Victory 
- Live Long 
- Peace
- Point Up 
- Fist 
- Custom gestures with text output
